¿Qué vamos a hacer?

En esta actividad analizaremos cómo funciona una estructura condicional en el lenguaje C para determinar si un usuario es mayor o menor de edad.

A continuación se muestra el código


    #include <stdio.h>

    int main() {
        int edad;
        printf("¿Cuántos años tienes? ");
        scanf("%d", &edad);

        if (edad >= 18) {
            printf("Eres mayor de edad.\n");
        } else {
            printf("Eres menor de edad.\n");
        }

        return 0;
    }

Explicación línea a línea

Línea 1: #include <stdio.h>
Incluye la librería estándar de entrada y salida.

Línea 3: int main() { ... }
Es la función principal y el punto de inicio de cualquier programa en C.

Línea 4: int edad;
Declara una variable llamada "edad". El tipo de dato int le avisa al compilador que esta variable solo guardará números enteros.

Línea 5: printf("¿Cuántos años tienes? ");
Muestra una pregunta en la pantalla del usuario para indicarle qué dato debe introducir.

Línea 6: scanf("%d", &edad);
Detiene el programa y espera a que el usuario escriba un número. El formato %d le dice que lea un entero, y el símbolo & (ampersand) indica que guarde ese valor directamente en el espacio de memoria asignado a la variable edad.

Líneas 8 a 12: if (edad >= 18) { ... } else { ... }
Es la estructura condicional. Evalúa si el número guardado en edad es mayor o igual a 18. Si se cumple (verdadero), ejecuta el bloque del if e imprime que es mayor de edad. Si no se cumple (falso), salta de inmediato al bloque del else e imprime que es menor de edad.

Línea 14: return 0;
Termina la ejecución de la función principal y le devuelve un cero al sistema operativo, lo que significa que el programa se ejecutó y cerró correctamente sin errores.
